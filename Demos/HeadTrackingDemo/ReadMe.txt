Head Tracking Demo - to be opened in Sally.

Open the Head Tracking Demo.cmp file in DAD.exe.

This file contains all the agents and state machines necessary to run the demonstration.

Go to the DemoStateMachine component in the Agent TreeView in DAD, and:
- Double-click on the LoadSM icon for the Agent List-View.	
  You should see a total number of 3 states appear in the window.

- Then, double-click on the Start icon in the Agent List-View.

The demo will start. The cameras will track whatever objected is segmented. Commonly Barney. The segmentation mask is found under the ColorSegmenter component under the Camera Agents.

- A visual display of the incoming and segmented images can be seen through manager book.

About the BH8-262 demonstration programs:

bhdemo45.exe
This application demonstrates many grasp types and features of the BarrettHand BH8-262 grasper.  The program runs in a loop and may be interrupted by pressing a key on the keyboard.


ForceResponse.exe
This application requires the STRAIN GAGE FORCE SENSOR option to be installed in the BarrettHand.  This program allows you control the movement of finger 2 by applying a force on finger 1.


MonitorStrain.exe
This application requires the STRAIN GAGE FORCE SENSOR option to be installed in the BarrettHand.  This program displays the present force sensor readings on-screen.  This is useful for force calibration and testing.

Component Development

By allowing the developer to separate the functionality of a solution into components, located where it is most logical for the solution, component-based design removes many of the constraints that used to hinder the deployment and maintenance of the solution. Use the documentation in this area to guide you to the component development techniques and tools that are right for your next solution.

About ActiveX Controls

Internet Development Index

This section introduces Microsoft ActiveX Controls and provides general guidelines for control developers. It includes information about run-time licensing for an ActiveX control, identifying an ActiveX control as safe for scripting or safe for code download, packaging an ActiveX control in a .cab file, and applying a test certificate to an ActiveX control.

In addition, this section includes information describing the differences between ActiveX Controls written for Microsoft Internet Explorer 3.0 and ActiveX Controls written for Internet Explorer 4.0.
